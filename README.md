# 🤖 ZARA — Rule-Based AI Chatbot

> DecodeLabs AI Internship 2026 | Project 1

## 📌 About
ZARA (Zero-Assumption Responsive Assistant) is a 
rule-based AI chatbot built entirely in Python.
No machine learning — just pure if-else logic and 
control flow to simulate human conversation.

## ✅ Features
- Continuous conversation loop
- Greeting & farewell detection
- Intent recognition using if-else logic
- Tells jokes, facts & motivational quotes
- Fallback response for unknown inputs
- Case-insensitive input handling
- Message counter milestone tracker

## 🛠️ Tech Stack
- Language : Python 3
- Libraries : None (pure Python)
- Concepts  : Control Flow, Decision Logic, Basic AI

## ▶️ How to Run
```bash
python zara_chatbot.py
```

## 💬 Supported Commands
| You type        | ZARA responds        |
|-----------------|----------------------|
| hello / hi      | Greeting             |
| how are you     | Status reply         |
| what is your name | Introduction       |
| joke            | Programmer joke      |
| fact            | Fun fact             |
| motivate me     | Motivational quote   |
| bye / exit      | Goodbye + stops      |
